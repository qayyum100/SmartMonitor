How to Set Up Email Alerts (Free)

3-step setup using EmailJS — no coding, no server

1
Create EmailJS Account

Go to emailjs.com → Sign up free (200 emails/month free)

2
Add Email Service

Go to Email Services → Add Service → Choose Gmail/Outlook → Copy the Service ID

3
Create Email Template

Go to Email Templates → Create New → Use variables: {{alert_type}}, {{message}}, {{time}}, {{to_email}} → Copy Template ID

4
Get Public Key

Go to Account → General → Copy Public Key → Paste all three above → Save!
